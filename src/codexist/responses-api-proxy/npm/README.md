# @openai/codexist-responses-api-proxy

<p align="center"><code>npm i -g @openai/codexist-responses-api-proxy</code> to install <code>codexist-responses-api-proxy</code></p>

This package distributes the prebuilt [Codexist Responses API proxy binary](https://github.com/openai/codexist/tree/main/codexist-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/codexist-responses-api-proxy.js --help
```

Refer to [`codexist-rs/responses-api-proxy/README.md`](https://github.com/openai/codexist/blob/main/codexist-rs/responses-api-proxy/README.md) for detailed documentation.
